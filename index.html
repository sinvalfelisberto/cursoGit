Comandos para definir usuário e senha:

  • git config --global user.name "sinvalfelisberto"
  • git config --global user.email "sinvalfelisberto@gmail.com"

Configurando seu repositório:

  • no site do github, criar um novo repositorio
  • depois, abrir o GitBash e digitar os comandos descritos abaixo:
  • git init
  • git add .
  • git commit -m "mensagem explicativa"
  • git branch -M main
  • git remote add origin git@github.com:seuusuario/seurepositorio.git
  • git push -u origin main
